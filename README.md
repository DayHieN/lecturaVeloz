
# speedRead

Herramienta que muestra un texto por pantalla, una palabra a la vez, dependiendo de las palabras por minuto (wpm) especificadas.
Para facilitar la lectura y leer rápido.

## Uso

``python speedRead <textfile> <wpm>``
