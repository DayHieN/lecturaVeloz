
# lecturaVeloz

Herramienta que muestra un texto por pantalla, una palabra a la vez, a la velocidad de las palabras por minuto (wpm) especificadas.
Para facilitar la lectura y leer rápido.

## Uso

``lecturaVeloz.py <textfile> <wpm>``
